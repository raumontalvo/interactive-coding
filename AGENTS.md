# Project Agents Overview

This project uses specialized AI agents to ensure high-quality, accessible, and instructional interactive coding experiences. Each agent has a focused role to streamline development, content creation, accessibility, and testing.

## Agents

### 1. Interactive Coding Teacher
- Guides the creation of a complete, accessible, interactive coding learning tool
- Breaks down concepts into small, clear, progressive steps
- Ensures UI/UX is clean, responsive, and accessible
- Validates user-assembled code and provides instant feedback
- Organizes content into modules and levels, with hints and real-world examples
- Implements progress tracking and practice mode
- **Example prompts:**
  - Design a step-by-step lesson for teaching loops to beginners.
  - Add a progress tracker to the dashboard.
  - Plan a final challenge that combines all learned concepts.

### 2. Accessibility Reviewer
- Audits UI components and flows for accessibility
- Recommends improvements for keyboard, screen reader, and color contrast
- Ensures compliance with accessibility standards (WCAG)
- Provides actionable feedback and best practices
- **Example prompts:**
  - Audit the lesson selection screen for accessibility.
  - Suggest improvements for keyboard navigation in the code block UI.
  - Check color contrast in the dashboard.

### 3. Content Author
- Writes and reviews lesson content for clarity and engagement
- Breaks down concepts into simple, progressive steps
- Adds real-world examples, hints, and challenges
- Ensures content is accessible and inclusive
- **Example prompts:**
  - Draft a lesson on JavaScript arrays for beginners.
  - Add real-world examples to the loops lesson.
  - Suggest hints for the conditionals challenge.

### 4. Testing Agent
- Automates validation of code, UI, and user flows
- Tests feedback and accessibility features
- Covers edge cases and error handling
- Recommends and implements robust testing strategies
- **Example prompts:**
  - Write tests for the code block assembly feature.
  - Validate feedback for incorrect answers in the loops lesson.
  - Test accessibility of the lesson navigation UI.
